Array.prototype.map()

        map() 메서드는 배열 내의 모든 요소 각각에 대하여 주어진 함수를 호출한 결과를 모아 새로운 배열을 반환합니다.

        구문
        arr.map(callback(currentValue[, index[, array]])[, thisArg])
        
        매개변수
        callback : 함수
        currentValue : 현재 요소
        index Optional : 인덱스

        반환 값 : 새로운 배열
